# 🐾 NovelReader

> SillyTavern 第三方扩展 —— 在酒馆中沉浸阅读本地小说

## ✨ 功能特性

- 📖 **TXT 小说导入** — 支持 UTF-8 / GBK 智能编码检测，大文件流式解析
- ✂️ **智能切章** — 正则表达式 / 固定字数 / 智能空行 / 不切章，四种模式
- 🎨 **10 套主题配色** — 樱花粉、嫩黄、淡绿、毛玻璃白/黑、夜樱、深海蓝、落日橘、薰衣草、摩卡棕
- 🐱 **悬浮球** — 可拖拽猫耳造型，支持自定义头像、头像框、尺寸调节
- 📚 **书架管理** — 列表/网格双视图、搜索高亮、自定义封面（图片/文字/Emoji）
- 🔤 **阅读器自定义** — 字体选择（系统/网络/本地上传）、背景图片、明暗调节
- 💾 **数据安全** — IndexedDB 本地存储，JSON 格式备份/还原，支持跨设备迁移
- 📱 **移动端适配** — 响应式布局，触摸拖拽，长按交互

## 📦 安装方式

### 方法一：通过 SillyTavern 扩展安装器（推荐）

1. 打开 SillyTavern，进入 **扩展** → **安装扩展**
2. 在输入框中粘贴本仓库地址：
https://github.com/yangtfei204-hub/NovelReader.git
3. 点击安装，刷新页面即可使用
